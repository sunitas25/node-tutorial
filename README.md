# Node Tutorial

## Description
This is the simple Node.JS tutorial.
 - Create a new folder
 - Open the folder in IDE
 - npm init
    Configure package details
 - Create src folder
    - Create app.js file
 - Install express
    npm install express
 - To avoid constant server restart use nodemon globally
   - npm install -g nodemon
   - Update package.json > Scripts:
        "start": "node src/app.js",
        "dev": "nodemon src/app.js"
    - Use npm run dev
 - Initialize git
    git init
 - Create .gitignore fle
    Add node_modules, .gitignore
